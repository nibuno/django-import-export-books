# 概要

本プロジェクトは、django-import-exportを試すリポジトリです。

- 本のモデルを定義してください
- 本のモデルをdjango-import-exportで出力します

## 技術スタック

- Django 5.2
- django-import-export

## 本のモデルについて

- あなたはプラウド出版に所属しています。
- 本には、次のようなものがあります。
  - [自走プログラマー](https://jisou-programmer.beproud.jp/)
  - [Pythonプロフェッショナルプログラミング 第4版](https://www.shuwasystem.co.jp/book/9784798070544.html)
- 本のモデルについては考えてください。

## なぜdjango-import-exportを使うのか

- 本のデータをエクスポートして、別の環境に移す必要があります。
- ただ、自分でエクスポート機能を0から作成するより、既存ライブラリを使った方が良いため、本ライブラリを使うことにします。
